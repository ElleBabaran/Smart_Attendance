# Smart_Attendance

Smart_Attendance is an automated attendance system that leverages computer vision and facial recognition to record student attendance efficiently. This project aims to simplify the process of tracking attendance while reducing errors associated with manual systems.

## Features

- **Real-time Face Detection:** Detects faces in real-time using a webcam or video feed.
- **Facial Recognition:** Identifies students based on pre-registered images.
- **Attendance Logging:** Automatically logs attendance with timestamps into a database.
- **User-Friendly GUI:** Simple and intuitive interface built using Tkinter.
- **Database Integration:** Stores attendance records and student details using SQLite (or other database systems).
- **Customizable:** Easy to adapt for different classroom or workplace environments.
